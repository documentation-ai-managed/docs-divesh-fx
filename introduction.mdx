## Quick Start

Get up and running with Divesh FX in minutes. Follow these steps to create your first documentation space.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com` and create your account with email or SSO.
  </Step>
  <Step title="Create a Space" icon="folder-plus">
    Click "New Space" and name it for your project, like "My Project Docs".
  </Step>
  <Step title="Add Your First Doc" icon="file-plus">
    Upload Markdown files or start writing directly. Use the editor to add components like cards and steps.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // Initialize Divesh FX client
  import { DiveshFX } from '@diveshfx/sdk';

  const client = new DiveshFX({ apiKey: 'YOUR_API_KEY' });
  await client.spaces.create({ name: 'My Project Docs' });
  ```
  ```python
  # Initialize Divesh FX client
  from diveshfx import DiveshFX

  client = DiveshFX(api_key='YOUR_API_KEY')
  client.spaces.create(name='My Project Docs')
  ```
</CodeGroup>
